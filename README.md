# RVFPGA_MSc_Project Updated for AY25-26
This is the Master Repo for the RVFPGA Project hosted at University of Sheffield. 
This contains a replica of the Imagination Technology University Program Materials, but these have been overlayed with additional materials

**Please do not change anything in this Repo.**
**For your project work, please create your own Branch!!!**

Here are some points for your quick start:

1. You can download the GitHub Desktop and connect the Repo to your local disk. See the following link: https://desktop.github.com/ Video for setting up your local repo: https://www.youtube.com/watch?v=8Dd7KRpKeaE
2. If you want to try the things on the video, please create your own Repo and try these out.
3. Please "**GitClone**" the project. Please DO NOT FORK IT! Forking will copy it in a new Repo to your GitHub page. You want to collaborate on the same GitHub Repo with other students (past and present).
4. Please DO NOT Change anything in the main branch until you are 100% sure that your new design works. It should always be deployable. Please create your own branch to start your work.
5. If you finish something and would like to submit the change to the main branch please talk toa member of staff first. If you push please don't forget to write a short description so that everybody knows what's going on.
6. Check out the support directory for slides and documents.

Please see this link for more details in using Github: https://medium.com/@jonathanmines/the-ultimate-github-collaboration-guide-df816e98fb67
